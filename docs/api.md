# API Endpoints

Base URL: http://localhost:3000

## Users

- GET /api/users
- POST /api/users
- GET /api/users/:id

## Projects

- GET /api/projects
- POST /api/projects
- GET /api/projects/:id
- GET /api/projects/:id/tickets

## Tickets

- POST /api/tickets
- PATCH /api/tickets/:id/status
- DELETE /api/tickets/:id
